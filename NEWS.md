# makicoint 1.0.0

* Initial CRAN release
* Implements the Maki (2012) cointegration test with multiple structural breaks
* Supports four model specifications:
  - Model 0: Level shift
  - Model 1: Level shift with trend
  - Model 2: Regime shift
  - Model 3: Regime shift with trend
* Allows 1 to 5 structural breaks
* Supports 1 to 4 regressors
* Multiple lag selection methods: t-sig, fixed, AIC, BIC
* Critical values from Maki (2012) Table 1
